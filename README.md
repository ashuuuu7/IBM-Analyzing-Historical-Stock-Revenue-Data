# Analyzing Historical Stock and Revenue Data

This project was completed as part of the IBM Data Science Professional Certificate and focuses on analyzing historical stock market performance and company revenue data using Python. The objective of the project is to demonstrate practical data acquisition, web scraping, data cleaning, and data visualization skills by working with real-world datasets from Tesla and GameStop.

The project begins with extracting historical stock price data using the yFinance library. Stock data for Tesla (TSLA) and GameStop (GME) was downloaded and stored in Pandas DataFrames for further analysis. The data was then cleaned and organized to prepare it for visualization and trend analysis.

In addition to stock market data, company revenue information was collected through web scraping techniques. Using the Requests and BeautifulSoup libraries, revenue tables were extracted from HTML webpages. The extracted data was cleaned by removing special characters such as commas and dollar signs, handling missing values, and converting data into a structured format suitable for analysis.

After preprocessing the datasets, exploratory analysis was performed to understand historical trends in stock prices and company revenues. The project then utilized visualization techniques to compare stock performance with revenue growth over time. Interactive graphs were generated using the provided make_graph() function to display the relationship between market valuation and business performance for both Tesla and GameStop.

Through this project, several important data science concepts were applied, including data collection from APIs, web scraping, data wrangling, data cleaning, exploratory data analysis, and visualization. The project also provided hands-on experience with working in Jupyter Notebooks and using industry-standard Python libraries for real-world analytical tasks.
